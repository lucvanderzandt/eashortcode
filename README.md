# Eashortcode
A WordPress plugin to easily add an "Add to chart" button in Visual Editor

## Summary
This plugin provides WordPress' Visual Editor with a button to easily add add_to_chart buttons to a blogpost or page. Users click on a small icon at the top right of Visual Editor showing an open box, choose a product, confirm et voil√°! Goodbye looking up and memorizing all the product SKUs or exact names!
